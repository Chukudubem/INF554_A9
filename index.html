Visualization Using Maps

Proportional Symbol Map, Chloropleth Map, Dot Map

We use the following maps to visually describe two different information:

1. A proportional symbol map showing the percentage of internet users in ten(10) countries in 2014

2. A chloropleth map showing the same information as in (1.) (Here the intensity of color is mapped to the percentage)

3. A dot map showing the active parks in Los Angeles County
